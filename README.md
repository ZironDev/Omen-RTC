# OMEN RTC

### Lightweight Discord Voice Client

A lightweight desktop client focused exclusively on Discord voice communication.

[Download Latest Release](https://github.com/ZironDev/Omen-RTC/releases/latest)

</div>

---

## About

OMEN RTC is a minimal desktop application built for users who only need Discord voice functionality without the overhead of a full-featured chat client.

The application focuses on providing a lightweight experience while remaining compatible with Discord's latest voice infrastructure.

---

## Features

- Lightweight desktop client
- Join Discord voice channels
- Speak in voice channels
- Minimal and clean interface
- Low memory usage
- Powered by Python
- Uses Opus for voice encoding
- Uses FFmpeg for audio processing
- Compatible with Discord's latest DAVE voice protocol

---

## Installation

1. Download **OMENRTCSetup.exe** from the latest release.
2. Run the installer.
3. Complete the installation.
4. Launch **OMEN RTC**.

---

## System Requirements

- Windows 10 (64-bit)
- Windows 11 (64-bit)
- Internet connection
- Discord account
